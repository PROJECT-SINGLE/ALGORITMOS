Projeto de Algoritmo — TheColor
📌 Visão geral
Este projeto foi desenvolvido com foco no uso de HTML, CSS e JavaScript, aplicando conceitos fundamentais de programação para tornar as páginas mais dinâmicas e interativas.

Tecnologias utilizadas
HTML — estrutura das páginas
CSS — estilização e interface
JavaScript — lógica e interatividade
Vetores (arrays)
JSON (manipulação de dados)
Lógica e funcionamento

O projeto aplica conceitos de programação para melhorar a dinâmica das páginas, como:

Uso de vetores para armazenar e organizar dados
Uso de JSON para manipulação e troca de informações
Estruturas de repetição (loops) para renderização automática de conteúdos
Manipulação dinâmica do conteúdo das páginas
Página “Conteúdos”

As páginas de Conteúdos seguem uma estrutura baseada em repetição no código.

Isso permite que:

Os conteúdos sejam carregados automaticamente
O sistema evite repetição manual de código
A manutenção fique mais simples e organizada

🧠 Quiz
O sistema de quiz foi desenvolvido com lógica de programação contendo:
Cálculos matemáticos para pontuação
Processamento de respostas do usuário
Geração de resultado final
Além disso, há uma função de porcentagem, que auxilia na:
👉 Escolha dos conteúdos mais adequados para o usuário com base no desempenho no quiz

Resumo
O projeto integra:
Estrutura web (HTML/CSS)
Lógica com JavaScript
Manipulação de dados com vetores e JSON
Estruturas de repetição para dinamismo
Cálculos matemáticos no quiz
Sistema de recomendação baseado em porcentagem

Resultado
O uso combinado dessas tecnologias torna o sistema mais:
Dinâmico
Organizado
Escalável
Interativo
